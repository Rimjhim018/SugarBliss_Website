SugarBliss!

SugarBliss is a modern and responsive ice cream website built using HTML, CSS, and Tailwind CSS. It includes sections like Home, Products, Menu, Specials, and Gallery. The design is made attractive with modern UI elements and hover effects.

🎯 Project Objective
Create a modern and visually appealing UI
Provide an easy product browsing experience
Ensure a fully responsive design across all devices
Improve UI/UX design skills

🛠️ Tools Used
HTML5, CSS3
Tailwind CSS
Google Fonts, SVG Icons
Images
Flexbox & Responsive Design

✨ Features
Responsive navigation bar
Product and menu cards
Hover effects and rating system
Special & Premium sections
Footer with location and social media icons
📱 Responsive Design
Supports mobile, tablet, and desktop devices
Built using Tailwind CSS and Flexbox
Mobile menu created with CSS

🎨 Design Techniques
Modern UI with a pink theme
Shadow and hover animations
Clean typography
Image effects
